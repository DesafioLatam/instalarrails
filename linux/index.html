Instalar Ruby on Rails en Linux (14.04)

Guía paso a paso para instalar Ruby on Rails en Linux

(En caso de cualquier duda o comentario, al final de cada página hay un foro para que puedes usar)

  1. Lo primero que debes hacer es abrir el terminal, para esto puedes apretar el botón "Windows" y luego escribir terminal

  2. Ahora vamos a instalar Ruby. Para esto debes copiar de a una cada una de las siguientes líneas y pegarlas en el terminal (shift+ctrl+v), luego presionar enter

    sudo apt-get update
    sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev

  3. Ahora vamos a instalar RVM, para manejar las versiones de Ruby (Ruby Version Manager), y vamos a aprovechar de configurar la versión de Ruby. Para esto, al igual que antes debemos copiar y pegar las siguientes líneas, de a una, en el terminal y luego darles enter.

    sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
    
    curl -L https://get.rvm.io | bash -s stable
    
    source ~/.rvm/scripts/rvm
    
    rvm install 2.3.1
    
    rvm use 2.3.1 --default
    
    ruby -v
    
    echo "gem: --no-ri --no-rdoc" > ~/.gemrc

    
    gem install bundler

  4. Ahora vamos a configurar Git, quien nos permitirá controlar las versiones de nuestros proyectos. 

    git config --global color.ui true
    
    git config --global user.name "Escribe tu nombre sin tildes, y letras como ø, ñ.."

    *El email a usar debe ser el mismo que luego usarás en Github 

    git config --global user.email "tucorreo@EMAIL.com"
    
    ssh-keygen -t rsa -C "tucorreo@EMAIL.com"

  5. Ahora vamos a instalar Rails

    curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
    
    sudo apt-get install -y nodejs
    
    gem install rails -v 4.2.7
    
    rails -v

    Tras este último comando, en el termina debería arrojarte: # Rails 4.2.7. Si no es así, significa que omitiste algún paso de la instalación

  6. Ahora vamos a configurar PostgreSQL 

    sudo sh -c "echo 'deb http://apt.postgresql.org/pub/repos/apt/ precise-pgdg main' > /etc/apt/sources.list.d/pgdg.list"
    
    wget --quiet -O - http://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | sudo apt-key add -
    
          sudo apt-get update
    
    sudo apt-get install postgresql-common
    
    sudo apt-get install postgresql-9.3 libpq-dev
