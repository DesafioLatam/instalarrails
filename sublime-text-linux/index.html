Instalar Sublime Text en Linux

Como desarrollador necesitarás habitualmente para tu código, un editor de texto. Uno bastante simple y con varias interesantes funcionalidades que recomendamos es Sublime Text

  1. En el terminal tienes que correr los siguientes comandos (Sublime Text 3) 

    
        sudo add-apt-repository ppa:webupd8team/sublime-text-3

      
    sudo apt-get update

      
            sudo apt-get install sublime-text-installer

  2. Perfecto! Ahora ideal si abres sublime text y creas un acceso directo, por de seguro lo vas a usar mucho :)
