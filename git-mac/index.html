Instalación de GIT

    Ahora debes instalar GIT en el terminal 

  1. brew install git

  2. Para verificar la instalación escribe en el terminal

    git --version

    El resultado aproximado debería ser git version 2.x.x Previous Step

  3. Ahora vamos a configurar Git, quien nos permitirá controlar las versiones de nuestros proyectos. 

    git config --global user.name "Escribe tu nombre sin tildes, y letras como ø, ñ.."

    *El email a usar debe ser el mismo que luego usarás en Github 

    git config --global user.email "tucorreo@EMAIL.com"
          
    ssh-keygen -t rsa -C "tucorreo@EMAIL.com"
