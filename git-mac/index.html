Instalación de GIT

    Primero verificamos que tengamos GIT instalado

    git --version

    Si obtenemos un error del tipo 

     commmand not found
    significa que no tenemos GIT instalado

  1. Si no tenemos GIT instalado

    En OSX: 

    brew install git

    En Ubuntu Linux: 

    sudo apt-get install git

  2. Ahora vamos a configurar Git, quien nos permitirá controlar las versiones de nuestros proyectos. 

    git config --global user.name "Escribe tu nombre sin tildes, y letras como ø, ñ.."

    *El email a usar debe ser el mismo que luego usarás en Github 

    git config --global user.email "tucorreo@EMAIL.com"
    ssh-keygen -t rsa -C "tucorreo@EMAIL.com"
