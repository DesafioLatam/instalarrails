Instalar Ruby on Rails en Windows

Guía paso a paso para instalar Ruby on Rails en Windows

(En caso de cualquier duda o comentario, al final de cada página hay un foro para que puedes usar)

  1. Lo primero que debes hacer es descargar lo necesario para la instalación de Ruby on Rails. Para esto debes entrar a RailsInstaller.org. y elegir una de las siguientes opciones (te recomendamos instales la última versión)

  2. Tras haber descargado el archivo, debes hacerle doble click comenzar con la instalación

  3. Tras lo anterior en la ventana de instalación, con tranquilidad avanza dándole "Next"

  4. Por último, tras avanzar en las ventanas instalación se abrirá el terminal y te pedirá en tu nombre y correo electrónico (no te preocupes, nadie verá tu email es para temas de configuración) .

    *Importante que cuandro escribas tu nombre no uses tildes ni letras como ñ, ø,...
