Instalar Ruby on Rails en Mac

Guía paso a paso para instalar Ruby on Rails en Mac

  1. Ahora debes instalar HomeBrew en el terminal 

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

    Si no te funciona, haz click acá y sigue las instrucciones

  2. Para verificar la instalación escribe en el terminal

    brew -v

    El resultado aproximado debería ser Homebrew 0.9.X

  3. Para asegurarte que está todo perfecto, escribe en el terminal

    4. brew doctor

    Si te pide hacer una actualización de brew, hazlo. Si te dice: “No such file or directory - /usr/local/Cellar” debes escribir lo siguiente en el terminal y dar enter 

    cd
    sudo mkdir /usr/local/Cellar

    Te pedirá tu clave (no podrás verla, es invicible)
