Instalación de RVM, Ruby y Rails

RVM es una herramienta que permite tener múltiples versiones de ruby instaladas en el sistema.

Nosotros instalaremos RVM para poder instalar Ruby

Las instrucciones oficiales para instalar RVM están aquí

Ahora vamos a instalar RVM, para manejar las versiones de Ruby (Ruby Version Manager), y vamos a aprovechar de configurar la versión de Ruby. Para esto, al igual que antes debemos copiar y pegar las siguientes líneas, de a una, en el terminal y luego darles enter.

  1. El primer paso es instalar la clave pública necesaria, para eso necesitaremos tener instalada la herramienta gnupg (para profundizar en el tema puedes ir aquí ). Para instalar la herramienta necesitamos ejecutar en el terminal:

    En OSX: 
     brew install gnupg gnupg2
    En Linux: 
     sudo apt-get install gnupg gnupg2

  2. Con gnupg2 instalado podemos agregar la llave pública necesaria ejecutando en el terminal: 

     gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3

  3. Con la llave pública en el llavero, podemos instalar RVM utilizando: 

    \curl -sSL https://get.rvm.io | bash -s stable

  4. Le pedimos a RVM que actualice los paquetes de nuestro sistema

     rvm requirements

  5. Tras lo anterior, cierra el terminal y vuelve a abrirlo, y escribe:

    rvm | head -n 1

  6. Ahora instalamos ruby

    rvm use ruby --install --default

    *En versiones antiguas, esto puede demorar hasta una hora.

  7. Ahora instalamos Rails 5

    gem install rails -v 5.0 --no-ri --no-rdoc

    Para verificar escribimos

    rails -v

    *El resultado esperado debería ser Rails 5.0.1

    8. *La instalación de Rails al igual que de ruby, puede demorar.
