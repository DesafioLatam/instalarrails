Instalar Ruby on Rails en Mac

Guía paso a paso para instalar Ruby on Rails en Mac

  1. Vamos a verificar que instalamos la correcta versión de ruby, para esto escribimos en el terminal

    ruby -v

    *Deberíamos tener la versión ruby 2.x, si no es así escribimos en el terminal

    rvm get stable

    rvm install 2.0.0 \
      --with-openssl-dir=$HOME/.rvm/usr \
      --verify-downloads 1

    rvm use --default 2.0.0

    *Tras esto deberíamos tener la versión que queremos (2.0), esto lo corroboramos con el comando

    ruby -v

  2. Vamos a verificar que instalamos la correcta versión de Rails, para esto escribimos en el terminal

    rails -v

    *Deberíamos tener la versión rails 4.x, si no es así y tenemos otra versión y/o tenemos un error, escribimos en el terminal

    gem update rails --no-ri --no-rdoc

    *Tras esto deberíamos tener la versión que queremos (4.x), esto lo corroboramos con el comando

    rails-v

  3. Ahora instalamos rails

    gem install rails --no-ri --no-rdoc

    Para verificar escribimos

    rails -v

    *El resultado esperado debería ser Rails 4.0.x

    4. *Notas adicionales

    La instalación de Rails al igual que de ruby, pude demorar. Si no se instala, prueba lo siguiente y luego reincia tu computador

    brew install apple-gcc42

    Si sigues sin avanzar prueba

    sudo xcodebuild -license


    sudo port upgrade outdated
    rvm reinstall 2.0.0
